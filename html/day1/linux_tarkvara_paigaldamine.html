Linux: Uue tarkvara paigaldamine¶

APT ja selle kasutamine¶

APT ehk Advanced Package Tool on programm, millega Linuxis mugavalt tarkvara paigaldada ja eemaldada. APT hoiab nimekirja erinevatest allikatest ehk repositooriumidest, laeb alla vajalikud tarkvarapaketid ning kopeerib failid õigetesse kohtadesse.

Tarkvaraallikate ja programmide nimekirja värskendamiseks on käsk:

sudo apt update

Ja uue programmi paigaldamiseks:

sudo apt install PROGRAMMINIMI

PROGRAMMINIMI asemel sista programm mida soovid paigaldada. Näiteks sudo apt install firefox või sudo apt install chromium-browser.

Märkus

sudo käivitab apt programmi administraatori õigustes.

Ülesanne 1¶

Paigalda eelneva näite põhjal Linuxisse programm nimega bastet.

Märkus

sudo kasutamisel küsitakse esimesel korral administraatori parooli. Sisesta t0ndik00bas ja vajuta ENTER klahvi.

Nõuanne

Linuxi terminali aknas ei ole parooli trükkimist näha. Seega, ära ehmu, kui sisestamisel tärne või mumme ei ilmu. Lihtsalt sisesta parool ja vajuta Enter.

Enne soovitud programmi paigaldamist kuvab APTkõikide pakkide nimekirja, ja küsib paigaldamiseks kinnitust. Sisesta y ja vajuta Enter klahvi.
